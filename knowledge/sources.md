# Source Rules

- Project links are navigation sources, not proof of a specific component API.
- Design-to-code work requires a node-specific Figma URL.
- Component props must be confirmed in Storybook or code before they are documented as facts.
- If Figma, Storybook and code disagree, report the conflict instead of choosing silently.
- If a source is missing, mark the field as `Unknown`, `Needs source` or `Needs review`.
