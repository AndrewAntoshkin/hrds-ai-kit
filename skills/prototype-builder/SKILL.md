# Prototype Builder

## Быстрый старт

1. Прочитай `../../knowledge/architecture.md`.
2. Компоненты — `../../knowledge/components-index.md`.
3. Паттерны — `../../knowledge/patterns-index.md`.
4. Примени `../../rules/components.md`, `../../rules/patterns.md`, `../../rules/accessibility.md`.
5. Вывод — композиция компонентов, не production-код (если не попросили иначе).

## План

- цель пользователя;
- выбор pattern;
- список components;
- states;
- tokens;
- missing sources;
- шаги реализации.
